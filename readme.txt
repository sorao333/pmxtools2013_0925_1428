ＰＭＸツール
blenderでＰＭＸモデルの剛体・ジョイントを扱うツールです
バイナリを扱う技術がないのでCVSでインポート・エクスポート
2.68なら確実ですが2.6以降なら動くとおもいます。
自分で使えりゃいいやってノリなので大雑把なツールです。
何か気付いた事があればブログの方に書いていただければと思います。
対処できるかどうか分かりませんが；

諸注意
当スクリプトを使用した事によって被った被害等の責任は負いかねますので
その点をご了承の上で御使用ください。
なにが起こるか分かりませんのでバックアップ等の対策をしてください。

入れ方
1.pmxtoolsフォルダをblenderのaddonsフォルダに入れてユーザー設定のアドオンタブで登録
2．内蔵テキストエディタにて__init__.pyを開いて起動
（スクリプトを修正しながら使うならこちらがやりやすいかも）
3.３Ｄビューのツールシェルフ（Tを押したら出てくる奴）にボタンが出てきます。

ユーザー設定の仕方やテキストエディタの使い方が
分からない場合はぐぐって下さい。



使い方

*最初にグループ　joint と rigid を作っておいてください。
前者はジョイント用後者は剛体用のグループです

:joint tool
	"骨に沿ってjoint設置"
	エディットモードで選択したボーンに沿ってジョイント配置
	1.先にgroup[joint]を作っておく
	2.エディットモードでボーンを選択する
	3.実行

"ジョイントデータ書き出し"
	ジョイントデータをCSVで書き出し
	書き出したＣＳＶはＰＭＤエディタで読み込めます。
	*ジョイントは必ず同一グループに登録すること
	*ジョイントを必ず一つ選択する事
	*カスタムプロパティが足りないとエラー

"jointファイル読み込み"
	pmdエディタで使用するＣＳＶファイルを読み込みます。
	1.先にgroup[joint]を作っておく
	2.実行
	*ファイルブラウザの左下にあるリネームチェックボックスにチェックを
	入れると名前の先頭に"joint_"がくっつきます。

:剛体tool　
" 骨に沿っ剛体を配置"　cube(箱)　sphere(球)　cylinder(カプセル)
	1.先にgroup[rigid]を作っておく
	2.エディットモードでボーンを選択する
	3.実行
	*日本語環境に変更してないとエラー
	英語で使いたい方はソースを読んで書替えてください。
	"回転コピー"って書いてある場所辺りです。ソース読めないって人は勉強してください。
	*剛体の位置や回転・サイズ等のデータは正確ですがイメージは完全に同じではありません。
	（サイズの縮尺がＭＭＤと違うため）
	ＰＭＤエディタで仕上げたほうが無難です。

"剛体データ読み込み＆書き出し"
	*ジョイントのと大体同じです。
	*剛体の書き出しはオブジェクトコンストレイントでの変形を適用して
	（ビジュアルトランスフォームを適用）実行して下さい。
	プロパティウインドウ上部のトランスフォーム「位置」「回転」「寸法」
	とオブジェクトのカスタムプロパティが書き出されるデータになります。

：その他のツール

"骨に沿って剛体チェーンを接合"
*選択した剛体とジョイントを接合します。blender上ではカスタムプロパティが書き換わるだけです。
*名前の順番に沿って連結します。
*剛体とジョイントは同じ数選択しないとエラーになると思います。
*ジョイントを先頭に↓の様に選択（・ジョイント□剛体）
　　　・□・□・□・□・□・□

"剛体ＡとＢをジョイントで結合"
＊Ａ＞Ｂと選択してから実行すると中間点にジョイントが作成され、
　ジョイントのカスタムプロパティ「剛体Ａ」にＡ、剛体Ｂ」にＢの名前が入ります。
　ジョイント名は”ＡtoＢ”になります。

・注意点	
	

	剛体”ＣＵＢＥ””ＣＹＬＩＮＤＥＲ”のボーンに沿っての配置の時ピボットポイントは（変形の中心）
	「中点」にしてください。

フォルダ内のtools.pyはblenderに登録されてないちょっとしたツールが入ってます。
